# 浏览器小游戏《躲避方块》需求文档

## 1. 项目简介

开发一个可直接在浏览器中运行的 2D 小游戏《躲避方块》。  
玩家通过键盘控制角色移动，躲避不断出现或下落的圆球障碍，尽可能获得更高生存时间或分数。

## 2. 项目目标

- 实现一个轻量、可玩、可迭代的网页小游戏原型。
- 不依赖任何前端框架，使用原生技术完成核心功能。
- 结构清晰，便于后续增加新玩法（如加速、道具、关卡）。

## 3. 目标用户

- 想快速体验小游戏的普通玩家。
- 用于前端基础练习或教学演示的开发者/学习者。

## 4. 核心玩法需求

### 4.1 玩家控制

- 玩家可通过键盘控制角色移动（方向键或 WASD）。
- 角色移动需限制在游戏区域内，不可超出边界。

### 4.2 障碍物机制（圆球）

- 敌对圆球按固定频率或随机频率生成。
- 圆球从上向下移动（或按设定方向移动）。
- 随游戏进行可逐步提升难度（生成频率变快、速度提升）。
- 障碍总量需随时间增加（例如：时间越久，同时存在的圆球数量越多）。

### 4.3 碰撞与失败判定

- 当玩家角色与任意圆球发生碰撞时，游戏结束（若有护盾则先消耗护盾并抵消一次碰撞）。
- 结束后显示“游戏结束”提示和最终得分/生存时间。

### 4.4 得分系统

- 基础规则：存活时间越长，分数越高。
- 实时显示当前分数。
- 可选：记录本局最高分（当前会话）。

### 4.5 游戏状态流转

- 初始状态：显示开始提示（如“按空格开始”）。
- 进行中：持续刷新画面并处理输入、移动、碰撞、计分。
- 暂停状态：点击“暂停”按钮可暂停，点击“继续”可恢复游戏进程。
- 结束状态：显示结果并支持重新开始。

### 4.6 技能系统（冲刺 + 护盾）

- 冲刺技能：玩家按 `Shift` 可触发短时间冲刺，提高移动速度并进入冷却。
- 护盾技能：拾取护盾道具后增加护盾层数，碰撞时优先消耗护盾，不立即结束游戏。
- HUD 需展示护盾数量与冲刺冷却状态（就绪/剩余秒数）。

### 4.7 道具掉落（减速 + 双倍分）

- 游戏过程中随机掉落道具球，玩家接触后触发效果。
- 减速道具：在持续时间内降低障碍移动速度。
- 双倍分道具：在持续时间内得分倍率提升为 2 倍。
- HUD 需展示道具效果是否激活及剩余时间。

### 4.8 波次机制（每 20 秒升级）

- 每 20 秒提升 1 波次，波次需在 HUD 实时显示。
- 波次升级后，障碍生成压力和速度上限随之提升。
- 叠加时间成长机制后，整体难度应保持可感知的阶梯式上升。

### 4.9 敌人行为分层

- 普通球：默认直线下落。
- 追踪球：在下落过程中会向玩家当前位置横向修正，形成追踪压力。
- 弹跳球：带有横向速度，触碰左右边界后反弹。
- 分裂球：下落到阈值后分裂为两个小球，提升局部密度与躲避难度。
- 不同类型敌人需在外观上可区分，便于玩家快速识别威胁。

### 4.10 Boss 事件（轻量）

- 每隔 60 秒触发一次 Boss 事件，持续 10 秒。
- Boss 事件在两种模式中随机或轮换触发：
  - 弹幕模式：短时间内高密度障碍弹幕。
  - 旋转激光区：出现可旋转的危险区域并持续施压。
- Boss 事件触发期间需有明显提示（文本、颜色或特效）区分于普通阶段。

### 4.11 视听反馈增强

- 连击音效：连续高风险操作（如连续擦弹/连续拾取）触发强化音效反馈。
- 低血红晕：危险状态（如无护盾且高压场景）出现屏幕边缘红晕提示。
- 击中前慢动作：在即将碰撞前触发约 0.1 秒慢动作，增强临场感。
- 轻微屏幕震动：受击、Boss 触发或高强度事件时触发短暂震动反馈。

### 4.12 主动攻击系统（子弹）

- 玩家可主动发射子弹（如按 `J` 或鼠标点击）攻击敌人。
- 子弹支持至少三个发射方向：正前方、左前方、右前方。
- 不仅正上方可发射，左前方和右前方也需可稳定发射并具备可见弹道差异。
- 子弹命中敌人后触发消灭判定并产生击中反馈（音效/特效）。
- 不同敌人类型可设置不同受击规则（如分裂球被击中后直接消失，或按阶段处理）。
- 子弹与敌人交互应影响得分（如击杀加分、连击加成）。
- 为避免滥发，需提供射击节奏限制（射速上限、弹药或冷却机制至少一种）。

## 5. 非功能需求

- 运行流畅：普通电脑浏览器下动画不卡顿（目标 60 FPS，允许波动）。
- 代码可读：HTML、CSS、JS 分文件管理，命名清晰。
- 兼容性：主流现代浏览器可正常运行（Chrome、Edge）。
- UI 风格现代化：整体需具备清晰层次、统一配色、按钮交互反馈与沉浸感视觉效果。

## 6. 技术选型

- 前端：HTML + CSS + JavaScript（无需框架）
- 开发工具：Cursor IDE
- 运行方式：直接用浏览器打开 `index.html`

## 8. 验收标准（MVP）

- 打开 `index.html` 可直接进入游戏页面。
- 玩家可通过键盘移动角色。
- 圆球障碍会持续生成并移动。
- 随存活时间增长，障碍数量明显增加（可感知到难度上升）。
- 点击暂停按钮可暂停，再次点击可继续，分数与障碍运动在暂停期间保持不变。
- 按 `Shift` 可触发冲刺，冷却结束后可再次使用，HUD 可见冲刺状态。
- 拾取护盾道具后可抵消一次碰撞，护盾数量变化正确显示。
- 减速与双倍分道具可随机掉落并生效，效果结束后自动恢复默认状态。
- 每 20 秒波次 +1，且每次波次升级后难度明显提高（速度/密度）。
- 场上会出现追踪球、弹跳球、分裂球，且行为符合定义并可感知区分。
- 每隔 60 秒触发 10 秒 Boss 事件（弹幕模式或旋转激光区），并有明显阶段提示。
- 连击音效、低血红晕、击中前 0.1 秒慢动作、轻微屏幕震动可在对应场景触发。
- 玩家可主动发射子弹并命中消灭敌人，击杀结果与得分反馈正确生效。
- 玩家可向正前、左前、右前三个方向发射子弹，三个方向均可命中敌人。
- 碰撞后能正确结束并显示结果。
- 支持重新开始游戏。
- UI 具备现代小游戏观感（例如：科技风背景、卡片化 HUD、高对比主按钮、悬浮反馈）。

## 9. 后续可扩展方向

- 增加移动端触屏控制。
- 增加音效、背景音乐、特效。
- 增加难度等级与排行榜（本地存储）。
