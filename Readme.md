# 光速闪击（浏览器小游戏）

一款基于原生 `HTML + CSS + JavaScript` 开发的 2D 网页小游戏。  
当前版本已做儿童友好适配：节奏更平缓、容错更高、提示更易理解，适合低年龄玩家体验。

## 快速开始

### 运行方式

- 方式一：双击 `start.bat`（Windows 一键启动）
- 方式二：直接浏览器打开 `index.html`

### 本地访问

- 默认地址：`http://localhost:8080`（通过 `start.bat` 启动时）

## 操作说明

- 移动：方向键 / `WASD`
- 冲刺：`Shift`
- 射击：长按 `J` 持续连发（多弹散射）
- 暂停/继续：界面中的“暂停”按钮
- 开始/重开：空格或按钮

## 儿童友好适配

- 默认难度下调：敌人速度与波次增长更平缓。
- 容错增强：护盾上限更高、道具刷新更频繁。
- 战斗反馈更积极：子弹射速与击中得分提升，鼓励持续互动。
- 压力控制：Boss 触发间隔延长，避免高强度频繁打断。
- 文案更温和：界面与结算提示使用更友好的表达方式。

## 核心玩法与系统

### 生存与难度成长

- 圆球敌人持续生成并移动
- 难度随时间提升（速度、密度、波次）
- 每 `20` 秒波次升级一次

### 敌人行为分层

- 普通球：直线下落
- 追踪球：横向跟随玩家
- 弹跳球：边界反弹
- 分裂球：到阈值后分裂成两个小球

### 主动攻击系统（子弹）

- 支持多方向发射：正前、左前、右前
- 每次开火为多弹散射（一次发射多颗子弹）
- 支持长按 `J` 持续射击
- 子弹速度已提升为基础版本的约 `2x`
- 命中敌人可消灭目标并获得击杀收益
- 长按 `J` 可持续连发，帮助儿童玩家更容易命中目标

### 技能与道具

- 冲刺（有冷却）
- 护盾（可抵消一次受击）
- 减速道具（降低敌人速度）
- 双倍分道具（短时得分加成）

### Boss 事件与反馈

- 每 `60` 秒触发一次轻量 Boss 事件，持续 `10` 秒
  - 弹幕模式
  - 旋转激光区
- 视听反馈增强：
  - 连击音效
  - 低血红晕
  - 击中前慢动作（约 0.1 秒）
  - 轻微屏幕震动
