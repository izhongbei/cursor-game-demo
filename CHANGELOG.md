# Changelog

本文件记录项目的重要变更，按日期和类型组织，便于追踪迭代。

## [2026-03-24]

### Added

- **战斗系统扩展**
  - 新增主动攻击系统（子弹发射与命中消灭敌人）
  - 新增多方向弹道（正前 / 左前 / 右前）
  - 新增多弹散射（单次开火多颗子弹）
  - 新增长按 `J` 持续连发能力（保留射速限制）

- **玩法机制扩展**
  - 新增技能系统：冲刺与护盾
  - 新增道具系统：减速、双倍分
  - 新增波次机制：每 20 秒升级
  - 新增敌人行为分层：追踪球、弹跳球、分裂球
  - 新增 Boss 事件（弹幕模式 / 旋转激光区）
  - 新增反馈机制：连击音效、低血红晕、慢动作预警、屏幕震动

- **界面与内容**
  - 新增左侧游戏说明面板（操作、道具、敌人与 Boss 说明）
  - 新增动漫主题图资源：
    - `anime-hero.svg`
    - `anime-enemy.svg`
    - `anime-scene.svg`
  - 新增并维护 `start.bat` 一键启动脚本

### Changed

- **视觉表现升级**
  - 游戏由 2D 观感升级为 3D 风格展示：
    - 场景透视与倾斜增强
    - 角色、敌人、道具、子弹、传送门采用深度缩放与 `translate3d`
    - 加强景深层次，提升立体感
  - 敌方球体增加旋转动效，强化动态表现
  - 为保证可见性，玩家角色改回稳定 2D 渲染并做边界可视钳制（防止 3D 透视导致越界或消失）

- **战斗参数调整**
  - 子弹速度提升为上一阶段约 `2x`
  - 连发与多发在原有冷却体系下协同工作
  - 五角星敌人加入敌人池并提升出现比例，局内更易出现
  - 敌人动漫贴图方案已撤回，恢复默认渐变圆球样式

- **儿童友好适配**
  - 默认难度下调（敌人速度、波次增长、Boss 触发节奏）
  - 提升容错（护盾上限增加、道具更频繁）
  - 提升正向反馈（射速与击中收益增强、提示文案更温和）

- **自动游玩行为**
  - 开局后支持自动持续发射子弹
  - 自动躲避威胁并保持持续巡航（避免长时间静止）
  - 护盾道具出现时提升追逐优先级，主动靠近并拾取

### Fixed

- 修复“按键已触发但无法发射子弹”的问题：
  - 输入事件正常接收
  - 子弹创建、更新、渲染、命中链路完整接入主循环
  - 命中后可正确消灭敌人并触发分数变化
- 修复玩家在 3D 透视下视觉越界与不可见问题：
  - 增加渲染钳制与安全边距
  - 最终采用玩家 2D 渲染保障稳定可见
